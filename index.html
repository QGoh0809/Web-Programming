What is currently being done well

With regards to the sites usability, it achieves a few industry standard Benchmarks for navigation, page design, content placement and an understanding of audience.

  • Navigation: The website successfully implemented a site map, and search facility based on keywords. Furthermore there is a functioning navigation bar implemented as a table utilising the <td> and <th> elements. (Figure 1)
  • Page Design: There is the successful implementation of a photo, as a logo, in the header of the site utilising the <img> element - providing brand and identity to the website. (Figure 1) Moreover, it should also be noted that the site effectively implements meta tags and alt tags. It also utilises a footer and header effectively.
  • Content Placement: The copy, at least for the ‘ above the line’ content on the landing page, was centred appropriately utilising a division tag with the appropriate use of margins, floats and inline properties to organise the text. (Figure 1)
  • Understanding of audience: The content would, in all likelihood, be of more interest to an academic or older community. With that in mind, the lack of graphics and visual stimulus illustrates an understanding of audience. The user’s visit the site for the unique information it holds - and don’t care for how it is delivered.

(Figure One)

What is currently not being done well

Although the site achieved some basic benchmarks, by enlarge it lacks effective consideration of Human Computer Interaction in all of the areas previously listed besides understanding of audience: navigation, page design, content placement.

  • Navigation: Although there is a navigation menu, sub menus like the Letters & Post Cards page does not contain any indication of which page you are on. This could be resolved with a sub class from the CSS utilised to highlight which page you are on in the main navigation (figure 2)
  • Page Design: To be frank, the design is dated and disengaging - with a lack of images and graphics to draw a broader audience beyond the academic and elderly community. Moreover, the use of colour, although contrasting, is also very dated. 
  • Content Placement: Although some pages are formatted correctly, most aren’t. For example, the Introduction page utilises a <span> tag instead of a simple <ul> lag.
  • Not Mobile friendly whatsoever, for example the graphics don't scale and the hamburger menu in non-reactive. (figure 3)

(Figure Two) / (Figure Three)

Redevelopment proposal

  • Develop CSS to help mitigate issues with indications, formatting for specific sections, logo spacing and scale etc.
  • Modernise colour scheme and logo.
  • Fix navigation features in sub-menus - maybe include them in the main navigation in the header.
  • Re-orientate copy to make sure the above the line content engages audiences further.
  • Implement more graphics, photos and multimedia
